# ElifTechSchool2019

## Overview
Backend for event system management.

## Getting Started

Install dependencies:
```sh
yarn
```

Create .env file
```bash
touch .env
```

Insert in .env file:
```
DB_HOST=<db host>
DB_NAME=<db name>
DB_PASSWORD=<db user password>
```

Start server:
```sh
# Start server
yarn start
```
